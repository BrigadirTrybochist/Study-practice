# Симулятор банківського рахунку

Простий консольний та графічний симулятор банківського рахунку (Python).

## 🚀 Швидкий старт

**1. Переконайтеся, що встановлена Python 3.8+:**
```bash
python --version
```

**2. Натисніть на один з батників:**
- `run_startup.bat` — консоль з меню вибору ролі ⭐ **РЕКОМЕНДУЄТЬСЯ**
- `run_simulator_gui.bat` — графічний інтерфейс
- `run_simulator.bat` — консоль у видимому CMD вікні

**✅ Готово! Ніяких докачувань не потрібно.**

---

## 📦 Що в репозиторії

- `SimulatorBanka.py` — основна консольна програма з меню вибору ролі (користувач/адміністратор)
- `SimulatorBankaGUI.pyw` — графічна програма (Tkinter) з таким же функціоналом
- `SimulatorBanka.json` — файл даних (рахунки і налаштування)
- `run_startup.bat` — запуск консолі (вибір ролі в консолі)
- `run_simulator.bat` — запуск консолі у видимому CMD вікні
- `run_simulator_gui.bat` — запуск GUI (без видимого CMD)
- `README.md` — цей файл
- `requirements.txt` — список залежностей (тільки Python 3.8+)

## Вимоги

**✅ Потрібна ТІЛЬКИ Python 3.8+ на Windows — нічого більше докачувати не потрібно!**

- Python 3.8+ (вже встановлена на більшості комп'ютерів)
- Tkinter (постачається з Python на Windows за замовчуванням)

### ⚠️ Важливо:
- Немає зовнішніх пакетів (`pip install` не потрібен)
- Все работает з стандартною бібліотекою Python
- Просто скачайте всі файли та натисніть батник

## Запуск

**Рекомендований варіант (консоль з вибором ролі):**
```batch
run_startup.bat
```
Запускає консольну версію, де ви можете вибрати роль (користувач або адміністратор).

**GUI (графічний інтерфейс):**
```batch
run_simulator_gui.bat
```
Запускає графічну версію з інтерфейсом користувача.

**Консоль у видимому вікні (для тестування):**
```batch
run_simulator.bat
```
